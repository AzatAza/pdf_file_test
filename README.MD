# Описание
Тестовое задание


## Как установить

Требуется `python3.9`

`pip install -r requirements.txt`

## Как запускать

1. Через Pycharm:
    * ПКМ по файлу
    * `Run 'pytest in file ..'`
    

2. Из консоли

   * `pytest` - запустит все
